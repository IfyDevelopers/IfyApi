# 😺 IfyApi
IfyApi - Library for simplifying the creation of multi-platform applications.

You can find information on using the library in the documentation: [Docs Site](https://api.ifydev.xyz/), if it does not work, then look at [Git Docs](https://github.com/IfyDevelopers/IfyApi/wiki).
## Installation

```zsh
npm install ifyapi
```
